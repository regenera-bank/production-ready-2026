## 🚀 INSTALAÇÃO (5 minutos)

### Passo 1: Clone e Setup

```bash
# Clone (ou use seus arquivos locais)
mkdir regenera-immortality
cd regenera-immortality

# Copie todos os arquivos gerados pelo Claude para esta pasta
# Estrutura esperada:
# ├── contracts/
# ├── scripts/
# ├── .github/
# ├── package.json
# ├── hardhat.config.js
# └── .env.example
```

### Passo 2: Instalar Dependências

```bash
# Instalar
pnpm install

# Ou se preferir npm
npm install

# Verificar
npx hardhat --version
```

### Passo 3: Configurar Variáveis de Ambiente

```bash
# Copiar template
cp .env.example .env

# Editar com suas credenciais
nano .env
```

Preencha:

```bash
# BLOCKCHAIN (obter em Infura.io)
POLYGON_RPC_URL=https://polygon-mainnet.infura.io/v3/YOUR_KEY
PRIVATE_KEY=0xYOUR_WALLET_PRIVATE_KEY
POLYGONSCAN_API_KEY=YOUR_POLYGONSCAN_KEY

# IPFS (obter em Infura.io e Pinata.cloud)
INFURA_PROJECT_ID=YOUR_INFURA_PROJECT_ID
INFURA_PROJECT_SECRET=YOUR_INFURA_SECRET
PINATA_API_KEY=YOUR_PINATA_KEY
PINATA_SECRET_KEY=YOUR_PINATA_SECRET

# AWS (opcional - para CDN)
AWS_ACCESS_KEY_ID=YOUR_AWS_KEY
AWS_SECRET_ACCESS_KEY=YOUR_AWS_SECRET
```

## 🎬 DEPLOYMENT AUTOMÁTICO (15 minutos)

### Opção A: Deploy Tudo de Uma Vez (Recomendado)

```bash
# Um único comando faz TUDO
npm run deploy:immortality

# Isso irá:
# 1. Compilar smart contracts
# 2. Deploy na blockchain
# 3. Upload para IPFS
# 4. Criar timestamps Bitcoin
# 5. Deploy badges/certificados
# 6. Ativar monitoramento forense
# 7. Gerar relatórios de verificação
```

**Tempo:** ~15 minutos  
**Saída:** Logs detalhados + arquivos em `/output/`

### Opção B: Deploy Passo a Passo (Avançado)

Se preferir controle total:

#### 1️⃣ Blockchain (3 min)

```bash
# Testnet primeiro (recomendado)
npm run blockchain:deploy:testnet

# Ver contrato no Mumbai Polygonscan
# https://mumbai.polygonscan.com

# Se OK, deploy mainnet
npm run blockchain:deploy:mainnet

# Registrar documentos
npm run blockchain:register
```

#### 2️⃣ IPFS (4 min)

```bash
# Upload todos os arquivos
npm run ipfs:upload

# Pin em múltiplos serviços
npm run ipfs:pin

# Gerar manifest
npm run ipfs:manifest

# Ver resultado
cat output/ipfs-manifest.json
```

#### 3️⃣ OpenTimestamps (2 min)

```bash
# Criar timestamps (Bitcoin)
npm run ots:stamp

# Arquivos .ots criados
# Aguardar 1-24h para confirmação
```

#### 4️⃣ GPG & GitHub (3 min)

```bash
# Setup GPG
npm run gpg:setup

# Seguir instruções na tela
# Copiar chave pública para GitHub

# Criar tag verificada
npm run git:tag

# Push
git push origin main --tags
```

#### 5️⃣ Badges (1 min)

```bash
# Gerar badges
npm run badge:generate
npm run vivo:generate

# Testar localmente
npm run badge:test

# Deploy (opcional)
npm run badge:deploy
```

#### 6️⃣ Forense (2 min)

```bash
# Ativar monitoramento
npm run forensic:start

# Ver logs
npm run forensic:logs

# Ver status
npm run forensic:status
```

## 🔧 TROUBLESHOOTING RÁPIDO

### Problema: "Insufficient funds"

```bash
# Verificar saldo
npx hardhat console --network polygon
> const balance = await ethers.provider.getBalance(deployer.address)
> console.log(ethers.utils.formatEther(balance))

# Precisa: ~0.1 MATIC (~$0.10 USD)
# Comprar em: https://quickswap.exchange ou CEX
```

### Problema: "IPFS timeout"

```bash
# Testar conectividade
curl -X POST "https://ipfs.infura.io:5001/api/v0/id" \
  -u "$INFURA_PROJECT_ID:$INFURA_PROJECT_SECRET"

# Se falhar, verificar credenciais
echo $INFURA_PROJECT_ID
echo $INFURA_PROJECT_SECRET
```

### Problema: "GPG key not found"

```bash
# Listar chaves
gpg --list-keys

# Se vazia, executar setup
npm run gpg:setup
```

### Problema: Deploy falhou no meio

```bash
# Limpar e recomeçar
npm run maintenance:clean
npm run maintenance:fresh-install
npm run deploy:immortality
```
