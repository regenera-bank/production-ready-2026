# Partner Developer Portal

Vite/React portal for Regenera Bank partner integrations.

## Features

- API documentation overview (Partner API v1 contract)
- Sandbox key management
- Webhook subscription and test delivery console

## Local development

```bash
pnpm install
# start facade in another terminal: pnpm --filter @regenera/partner-api-facade start:dev
pnpm dev
```

Portal: `http://localhost:3400`
