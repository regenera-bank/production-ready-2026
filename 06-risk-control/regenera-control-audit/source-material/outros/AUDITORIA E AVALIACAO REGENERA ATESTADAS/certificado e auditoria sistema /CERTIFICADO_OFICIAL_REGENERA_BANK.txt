    2. Ir para aba "Read Contract"

       → Deve retornar: true + dados do documento
    5. Executar função: isCertificateValid(1)
       → Deve retornar: true

MÉTODO 2: Verificação via Hash SHA-256 (Mais Técnica)
    
    Linux/macOS:
        sha256sum REGENERA_BANK_AUDITORIA_TECNICA_V1.md
        # Output deve ser: 49947b89f72ab62dd9da63e70f9c1bd7dc07a9fbbeef556c9a1336309d34595b
    
    Windows PowerShell:
        Get-FileHash REGENERA_BANK_AUDITORIA_TECNICA_V1.md -Algorithm SHA256
        # Output deve ser: 49947b89f72ab62dd9da63e70f9c1bd7dc07a9fbbeef556c9a1336309d34595b
    
    Comparar o hash calculado com o hash registrado na blockchain.
    Se MATCH = documento é autêntico e não foi modificado.

MÉTODO 3: Verificação via IPFS (Distribuído)
    
    1. Acessar qualquer gateway IPFS:
       https://ipfs.io/ipfs/[DOCUMENT_CID]
    2. Download do arquivo
    3. Calcular hash SHA-256
    4. Comparar com blockchain
